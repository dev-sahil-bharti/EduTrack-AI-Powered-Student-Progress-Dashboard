## ✨ Features

- 🎯 **Set & Track Goals**: Keep an eye on weekly subject targets and stay ahead of your schedule.
- ⏱️ **Log Daily Progress**: Log study minutes effortlessly to maintain your streak.
- 🤖 **AI-Powered Insights**: Receive personalized, actionable study tips generated by Google's Gemini AI.
- 📊 **Dynamic Data Visualization**: See your progress come to life in beautiful, interactive charts.
- 🔐 **Secure Authentication**: Quick, safe, and easy login via Google OAuth integration.
- 🌙 **Modern Dark Mode**: A sleek, eye-friendly user interface optimized for late-night study sessions.

## 🚀 Quick Start Guide

Want to run this codebase locally? It's incredibly easy! Just follow these simple steps:

### 1️⃣ Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### 2️⃣ Clone and Install
Open your terminal, ensure you are inside the frontend project folder, and install the required dependencies:
```bash
npm install
```
*(This downloads all the packages listed in `package.json` directly onto your machine).*

### 3️⃣ Configure Environment
Create a `.env` file in the root of the project folder and add your specific environment variables. You will need your backend URL and Google Client ID:
```env
# Backend API URL (Default local)
VITE_API_URL=http://localhost:5000/api

# Google OAuth Client ID for Authentication (if applicable)
VITE_GOOGLE_CLIENT_ID=your_google_client_id_here
```

### 4️⃣ Start Development Server
Fire up the Vite development server:
```bash
npm run dev
```

🎉 **Boom!** Your application is now running. Open your browser and navigate to **[http://localhost:5173](http://localhost:5173)** to see the magic.

## 📂 Project Structure

Navigating a new codebase can be intimidating, but we've kept things simple, modern, and organized based on standard React patterns:

```text
frontend/
├── 📁 src/
│   ├── 🗂️ api/          # Axios interceptors and global API configurations
│   ├── 🗂️ components/   # Reusable UI building blocks (Navbar, Sidebar, Cards)
│   ├── 🗂️ context/      # Global state management (e.g., AuthContext for login)
│   ├── 🗂️ pages/        # Main application views (Dashboard, Login, Insights)
│   ├── 🗂️ services/     # Feature-specific server API functions
│   ├── 📄 App.jsx       # Application router and layout configuration
│   └── 📄 main.jsx      # The React entry point (where the app boots up)
├── 📄 vite.config.js    # Vite bundler configuration
└── 📄 package.json      # Dependencies and project scripts
```
