
## What is EduTrack?

**EduTrack** is an AI-powered student dashboard designed to make learning manageable, trackable, and engaging. It provides a sleek, dark-mode user interface where students can:
- **Set & Track Goals**: Keep an eye on weekly subject targets.
- **Log Daily Progress**: Log study minutes effortlessly.
- **Get AI Feedback**: Receive personalized actionable tips generated by Google's Gemini AI.
- **Visualize Data**: See progress in beautiful charts and graphs.

## The Tech Stack

We've chosen a modern, lightweight, and incredibly fast stack to ensure a fluid user experience.

| Technology      | Why we use it |
|-----------------|---------------|
| **React 19**    | Component-based architecture for scalable UI. |
| **Vite**        | Ultra-fast module bundler and dev server. |
| **TailwindCSS** | Clean, responsive utility-first styling (No bloated CSS!). |
| **Recharts**    | Awesome, easy-to-read dynamic data charts. |
| **Framer Motion**| Smooth, delightful micro-animations & transitions. |
| **Axios**       | Clean and interceptor-friendly HTTP requests. |

## Quick Start Guide

Want to run this codebase locally? It's super easy! Just follow these steps:

### 1️⃣ Ensure Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### 2️⃣ Install Dependencies
Open your terminal, ensure you are inside the `client` folder, and run:
```bash
npm install
```
*(This downloads everything needed from the `package.json` file into your computer).*

### 3️⃣ Configure Environment
If you need to connect to the backend, create a file named `.env` right here inside the `client` folder:
```env
VITE_API_URL=http://localhost:5000/api
```

### 4️⃣ Start the Dev Server
Run the Vite development server:
```bash
npm run dev
```
🎉 **Boom!** Your application is now running. Open your browser and go to **[http://localhost:5173](http://localhost:5173)**.

## 📂 Understanding the Folder Structure

Don't feel overwhelmed! Here is exactly where everything lives:

```text
client/
├── 📁 src/
│   ├── 🗂️ api/          # Axios setup (how we talk to the backend)
│   ├── 🗂️ components/   # UI building blocks (Navbar, Sidebar, Buttons)
│   ├── 🗂️ context/      # Global state (We handle user login here!)
│   ├── 🗂️ pages/        # Entire pages (Dashboard, Login, Insights)
│   ├── 🗂️ services/     # Direct API call functions
│   ├── 📄 App.jsx       # The main router (defines all website URLs)
│   └── 📄 main.jsx      # The starting point of the React app
├── 📄 vite.config.js    # Settings for the Vite bundler
└── 📄 package.json      # Complete list of all packages we installed
```
